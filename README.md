## 霞鹜文楷GB简介

霞鹜文楷GB是一款严格遵循中国国家字形标准的开源中文字体，基于日本 FONTWORKS 公司 Klee One 字体改进而来，为个人和企业用户提供完全免费的商用字体解决方案。

这款字体不仅美观实用，更重要的是完全符合教育用途标准，是现代中文排版的理想选择。8105 个汉字支持，完整覆盖《通用规范汉字表》所有汉字。

### 字体特性

- **国标字形规范**：严格按照《通用规范汉字表》和《通用规范汉字笔顺规范》进行字形优化
- **符合书写习惯**：完全符合楷体书写习惯，消除传统印刷字体的"直点"和"断笔"特征
- **多字重选择**：提供 Light、Regular、Medium 三种字重版本
- **配套等宽字体**：配套等宽字体版本，满足编程和排版需求
- **免费商用**：采用 SIL Open Font License 1.1 授权协议，个人和企业均可自由使用

### 适用场景

- **教育出版领域**：义务教育教材和少儿读物排版、字帖和书法练习材料制作、教育类软件和应用程序界面设计
- **商业设计应用**：企业宣传材料和品牌设计、网页设计和移动应用界面、印刷品和包装设计
